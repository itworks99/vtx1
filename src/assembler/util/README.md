# VTX1 Assembler Utility Module

This directory contains utility functions and shared components for the VTX1 assembler.

## Purpose

The utilities module provides common functionality used across different parts of the assembler:
- Symbol table management
- Error handling and reporting
- Binary file I/O
- Ternary number conversion utilities
- Configuration handling

## Implementation

These utilities are designed to be reusable across the different assembler phases and components.
